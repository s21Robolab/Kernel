Privacy Policy

Kernel Discord Bot

Last updated: 2026

1. Data We Collect

Kernel collects and processes only the minimum data required for its functionality, including:

Discord User ID

Discord Server (Guild) ID

Discord Role ID

2. Data We Do NOT Collect

Kernel does not collect or store:

message content;

private messages;

email addresses;

passwords or authentication credentials;

IP addresses.

3. How Data Is Used

Collected data is used solely to:

verify users;

assign, update, or remove Discord roles;

ensure correct operation of the Bot.

4. Data Storage

Data may be stored temporarily or persistently for operational purposes.

No data is used for tracking, analytics, or advertising.

5. Data Sharing

Kernel does not share collected data with third parties.

6. Data Security

Reasonable technical measures are taken to protect stored data. However, no system can be guaranteed to be completely secure.

7. Data Removal

If the Bot is removed from a server, associated server-related data may be deleted automatically.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the Bot indicates acceptance of the revised policy.

9. Contact

If you have questions regarding this Privacy Policy, contact the developers via:
